# Flower

- Fluxo / Follow / Follower / Following / PERT / Gantt
- Métodológia para detectar qualidade de um processo.
- Cloudflare Workers

## Instalação:
- Node 18
- NPM

```bash
nvm use 18
npm ci
npm run pack
```
